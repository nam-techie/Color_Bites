# COLOR BITES 🎨✨

> **Khám phá thế giới ẩm thực qua lăng kính tâm lý học màu sắc**

COLOR BITES là nền tảng web tiên tiến sử dụng AI để phân tích tâm trạng thông qua trắc nghiệm màu sắc, từ đó đưa ra gợi ý cá nhân hóa về món ăn, đồ uống và phong cách thời trang phù hợp với cảm xúc của bạn.

## 🌟 Tính năng nổi bật

### 🧠 **Trắc Nghiệm Màu Sắc AI**
- Phân tích tâm trạng tiên tiến thông qua khoa học màu sắc
- Kết quả cá nhân hóa trong vòng 30 giây
- Không cần đăng ký, bảo mật tuyệt đối

### 🍽️ **Gợi Ý Ẩm Thực Thông Minh**
- Món ăn phù hợp với tâm trạng hiện tại
- Công thức nấu ăn chi tiết
- Danh sách nhà hàng gần bạn tại Thủ Đức

### 👗 **Tư Vấn Thời Trang**
- Phong cách trang phục theo cảm xúc
- Gợi ý màu sắc và phụ kiện
- Danh sách cửa hàng thời trang

### 🗺️ **Bản Đồ Khám Phá**
- Khám phá địa điểm ẩm thực Thủ Đức
- Đánh giá và review từ cộng đồng
- Tích hợp Google Maps

### 👥 **Cộng Đồng Sôi Động**
- Chia sẻ món ăn và trải nghiệm
- Hệ thống like, comment, reply
- Hashtag và tìm kiếm thông minh

## 🚀 Cài đặt và chạy

### Yêu cầu hệ thống
- **Node.js** 18+ 
- **npm** hoặc **yarn**
- **Git**

### 1. Clone repository
```bash
git clone https://github.com/your-username/color-bites.git
cd color-bites
```

### 2. Cài đặt dependencies
```bash
cd frontend
npm install
```

### 3. Chạy development server
```bash
npm run dev
```

🎉 **Ứng dụng sẽ chạy tại:** `http://localhost:5173`

### 4. Build cho production
```bash
npm run build
npm run preview
```
